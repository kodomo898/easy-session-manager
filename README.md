# easy-session-manager

Easily start an AWS Systems Manager Session Manager.

 - Example: ssm [list/start [instance_id]/help]

 - list  :Get Available Instance List
 - start :Session start example: ssm start [instance_id]
 - help  :it is help

# How to use MFA
